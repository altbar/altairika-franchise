# Шаг 8. Figma MCP

Что даёт: Claude может читать наши общие Figma-файлы — забирать промо-материалы, описания макетов, размеры, цвета, тексты. Полезно для: подготовки постов с готовыми картинками, описания материала перед публикацией, сборки SMM-карусели из шаблонов.

## Что использовать

`figma-developer-mcp` (он же framelink) — официальный MCP-коннектор, читает Figma по API через токен.

## Установка

### Шаг 1 — получи Figma API token

1. Зайди в Figma → правый верхний угол → твой аватар → **Settings**
2. Перейди в **Security** → раздел **Personal access tokens**
3. Жми **Generate new token**
4. Имя: `claude-mcp`. Срок: 90 дней (для безопасности — потом обновишь)
5. Скоупы: **File content (read-only)** — этого достаточно
6. Скопируй токен сразу — больше его не покажут

### Шаг 2 — подключи MCP в Claude

Открой `~/.claude/settings.json` (Mac) или `%USERPROFILE%\.claude\settings.json` (Windows) и добавь в `mcpServers`:

```json
{
  "mcpServers": {
    "figma-dev": {
      "command": "npx",
      "args": ["-y", "figma-developer-mcp", "--stdio"],
      "env": {
        "FIGMA_API_KEY": "figd_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
      }
    }
  }
}
```

Перезапусти Claude (выйди из CLI, запусти `claude` снова).

### Шаг 3 — получи доступ к корпоративным файлам Altairika

Напиши Константину в TG (@altbar) **email, под которым ты залогинена в Figma**. Тебе откроют доступ к общему пространству Altairika — там макеты постов, промо, баннеры, презентации.

## Проверка

Возьми ссылку на любой файл из общего пространства Altairika (например, шаблон поста). Скопируй URL вида `https://www.figma.com/file/AbCdEf123/Promo-Templates?node-id=1-2`.

В Claude:
> Посмотри этот файл Figma: <ссылка>. Опиши, что там есть, какие фреймы, какие тексты.

Claude должен достать структуру файла, имена фреймов, текстовые слои.

## Полезные сценарии

- **"скачай этот фрейм как PNG"** — Claude использует `download_figma_images` и сохранит локально
- **"перепиши заголовок в этом макете под событие 'летний интенсив'"** — даст готовый текст под бренд-тон
- **"какие промо-баннеры есть для дней рождения?"** — пройдёт по корпоративным файлам Altairika и составит список

## Безопасность

- Токен Figma — секрет. Не коммить `settings.json` в git если в нём есть `FIGMA_API_KEY`
- Если потерял доступ — сходи в Figma Settings → Security → отзови токен и создай новый
- Не давай Claude задач массового скачивания всего пространства — это нагружает Figma и может привести к rate-limit
