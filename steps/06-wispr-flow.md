# Шаг 6. Wispr Flow — голосовой ввод

Что даёт: говоришь — оно вставляет текст в активное окно. Включая prompt в Claude. Драматически ускоряет работу — длинный контекст и развёрнутые промпты быстрее надиктовать чем напечатать. Wispr убирает "э-э", повторы, расставляет знаки препинания.

После 2-3 дней использования становится рефлексом — почти не возвращаешься к печати для длинных промптов.

## Установка

### Mac

```bash
brew install --cask wispr-flow
```

или скачай с https://wisprflow.ai → запусти .dmg → перенеси в Applications.

### Windows

Скачай `.exe` с https://wisprflow.ai → запусти инсталлер → следуй мастеру.

## Setup (одноразово)

1. Запусти Wispr Flow (Mac: cmd+space → Wispr Flow; Windows: меню Пуск)
2. Войди по email — придёт код подтверждения
3. **Дай разрешения:**
   - **Микрофон** — обязательно
   - **Accessibility** (Mac: System Settings → Privacy & Security → Accessibility → добавь Wispr Flow с галочкой)
   - На Windows может попросить разрешение на симуляцию ввода — соглашайся
4. Hotkey по дефолту:
   - **Mac**: hold **Fn** (нижний левый угол клавиатуры)
   - **Windows**: hold **Right-Alt** или **Win**
5. В настройках можно сменить hotkey на любой удобный (Wispr Flow → Settings → Shortcuts)

## Использование с Claude в терминале

1. Сфокусируй окно терминала, где запущен `claude`
2. **Удерживай hotkey** — пошёл индикатор записи (значок Wispr в трее замигает)
3. **Говори в обычном темпе**, не торопись. Wispr сам уберёт "э-э", повторы, расставит знаки препинания.
4. **Отпусти hotkey** — текст вставится в prompt на месте курсора
5. Проверь визуально и нажми Enter

## Когда диктовать vs печатать

| Тип ввода | Что выбрать |
|-----------|-------------|
| Длинный промпт (≥3 предложения, контекст, описание задачи) | Диктуй |
| Короткая команда / правка / `/help` / `/clear` | Печатай |
| Микс | Начни печатать, дозиктуй длинный кусок, дочини печатью |

## Тарифы

Wispr Flow бесплатен с лимитом ~2000 слов в день. Для большинства франчайзи этого хватает. Если упираешься в лимит — есть платная подписка ($15/мес unlimited), но обычно не нужна.

## Тонкости

- **Не работает в защищённых полях** — пароли, secure-input в банковских клиентах. Wispr намеренно блокируется системой, это безопасно.
- **Если терминал странно вставляет текст** — проверь раскладку клавиатуры. Wispr эмулирует ввод символов, и при русской раскладке могут вставляться кириллические символы вместо латинских. Решение: переключи раскладку на английскую перед диктовкой команд, на русскую — перед диктовкой текстов.
- **Голосовое окружение:** Wispr хорошо справляется с тихим офисом. На улице с ветром или в шумном кафе — будет ошибаться. Микрофон ноута достаточно, наушники с микрофоном (AirPods и т.п.) — лучше.
