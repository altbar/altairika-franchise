# Шаг 11. Exa MCP — web search

Что даёт: качественный поиск в интернете и быстрое чтение веб-страниц. Это работает заметно лучше встроенного web-поиска Claude — Exa специально натренирован на семантический поиск и возвращает релевантные результаты, не SEO-мусор. Полезно для: исследования конкурентов в твоём городе, мониторинга упоминаний Altairika, поиска идей для постов и активностей.

## Регистрация и API-ключ

### Шаг 1 — зарегистрируйся на Exa

1. Открой https://dashboard.exa.ai/login (через VPN — нужно)
2. Войди через Google (рабочая или личная почта — на твой выбор)
3. Подтверди email если попросят

### Шаг 2 — получи API-ключ

1. На дашборде слева выбери **API Keys**
2. Жми **Create new key** → имя `claude-mcp` → создать
3. Скопируй ключ (формат: `xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx`) — больше его не покажут целиком

### Шаг 3 — про бесплатные $10

При регистрации Exa выдаёт **$10 бесплатных credits**. Этого хватает надолго:
- Базовый поиск: ~$0.005 за запрос → **2000 запросов**
- Сложный (с контентом и LLM-rerank): ~$0.025 за запрос → **400 запросов**

Среднестатистический франчайзи делает 3-10 поисков в день. **$10 хватит на 4-6 месяцев** активного использования. Когда закончатся — пополнишь карточкой ($10 минимум).

Контролировать расход: Dashboard → **Usage** → видно сколько потрачено и сколько осталось.

## Установка MCP

### Шаг 1 — подключи в Claude

Открой `~/.claude/settings.json` и добавь:

```json
{
  "mcpServers": {
    "exa": {
      "command": "npx",
      "args": ["-y", "exa-mcp-server"],
      "env": {
        "EXA_API_KEY": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx"
      }
    }
  }
}
```

Перезапусти Claude.

## Проверка

Спроси Claude:
- **"найди через exa последние новости про VR-аттракционы для детей в России"**
- **"найди через exa, что пишут про Altairika в 2026 году"**
- **"найди через exa конкурентов VR-развлечений для детей в Барнауле"**

Claude должен использовать `mcp__exa__web_search_exa` или похожий tool и вернуть результаты со ссылками и сниппетами.

## Полезные сценарии для франчайзи

- **"что нового в индустрии детских VR-развлечений в этом квартале?"** — мониторинг тренда
- **"найди статьи про развитие ребёнка через VR — 5 свежих, для поста в TG-канал"** — контент-идеи
- **"посмотри сайты конкурентов в моём городе и составь сравнительную таблицу"** — конкурентный анализ
- **"найди 10 идей активностей на летние каникулы для детей 6-10 лет"** — программирование
- **"выкачай содержимое этой статьи и сделай краткую выжимку"** — `web_fetch_exa` берёт страницу целиком

## Безопасность

- API-ключ Exa — секрет. Не коммить `settings.json` если в нём ключ.
- Если ключ утёк — Dashboard → API Keys → Revoke → создай новый.
- Если кто-то начнёт использовать твой ключ — увидишь скачок в Usage. Лимит расхода поставь в Dashboard → Settings → Spend limit (например, $20/месяц).
