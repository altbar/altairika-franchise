# Шаг 7. Telegram MCP

Что даёт: Claude может читать твои Telegram-чаты, отправлять сообщения, искать по истории, реагировать на сообщения. Полезно для: ответа родителям прямо из Claude, поиска по переписке "что я обещал заказчику в апреле", автоматизации рутинных ответов.

## Что использовать

`telethon-telegram` — MCP, который работает через **твой личный Telegram-аккаунт** (не бот). Полный доступ ко всему, что видишь ты в приложении.

> Важно: это твой **личный** аккаунт, не бот. Делает всё, что делаешь ты руками. Спам или массовую рассылку Telegram банит — не используй для холодных рассылок.

## Установка

### Шаг 1 — получи api_id и api_hash

1. Зайди на https://my.telegram.org/auth (через VPN)
2. Введи свой номер телефона, который привязан к Telegram → получи код в Telegram
3. Войди → "API development tools"
4. Заполни форму (App title: "claude-mcp", short name: "claude_mcp", platform: Desktop)
5. Скопируй `api_id` (число) и `api_hash` (строка). Они одноразово показываются — сохрани сразу.

### Шаг 2 — установи MCP-сервер

Сначала проверь Python:
```bash
python3 --version    # Mac
python --version     # Windows
```
Должно быть 3.10+. Если нет — поставь с https://www.python.org/downloads/

Установка MCP:
```bash
pip install telethon-mcp
```

(или другая команда — зависит от выбранного пакета; уточню в следующей итерации кита)

### Шаг 3 — пропиши в Claude config

Открой `~/.claude/settings.json` (Mac) или `%USERPROFILE%\.claude\settings.json` (Windows) и добавь в `mcpServers`:

```json
{
  "mcpServers": {
    "telethon-telegram": {
      "command": "python3",
      "args": ["-m", "telethon_mcp"],
      "env": {
        "TELEGRAM_API_ID": "12345678",
        "TELEGRAM_API_HASH": "abc123def456...",
        "TELEGRAM_PHONE": "+79991234567"
      }
    }
  }
}
```

### Шаг 4 — первый запуск (авторизация)

Запусти Claude → попроси: **"подключись к telegram и отправь мне в Saved Messages 'тест'"**.
- Claude инициирует подключение
- Telegram пришлёт код подтверждения в твой основной аккаунт
- Скажи Claude этот код — авторизация запомнится в локальном session-файле, повторно код запрашиваться не будет

## Проверка

Спроси Claude: **"покажи мои последние 5 чатов в telegram"** или **"найди в моих чатах сообщения от руководителя за последний месяц"**.

## Безопасность

- Session-файл (`~/.claude/telethon.session` или похожий) даёт полный доступ к твоему Telegram. Не коммить его в git.
- Если потерял ноут — сразу зайди в Telegram → Settings → Devices → терминируй сессию "claude-mcp"
- Не давай Claude задач "массовая рассылка по списку" — Telegram забанит
