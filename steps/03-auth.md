# Шаг 3. Авторизация Claude

Claude Code нужна подписка от Anthropic. Бесплатно работать он не будет — каждый ответ Claude стоит компании денег.

## Какая подписка нужна

| План | Цена | Что даёт | Кому подойдёт |
|------|------|----------|----------------|
| **Claude Pro** | $20/мес | Веб-чат + Claude Code (ограниченный лимит) | Базовый старт, попробовать |
| **Claude Max** | $100/мес (5x) или $200/мес (20x) | Высокий лимит токенов, доступ к Opus 4.6 в Claude Code | Если будешь активно использовать |
| **API key** | Pay-per-use | Только Claude Code, без чата | Если хочешь точный контроль расходов |

**Рекомендация для франчайзи:**
- Старт — **Claude Pro $20**. Хватит на пару часов в день.
- Если плотно зайдёт через месяц — переходи на **Claude Max $100** (это 5-кратный лимит и доступ к самой мощной модели Opus).
- Учитывай: оплата в долларах, нужна международная карта или сервис-посредник.

> УК Altairika при необходимости может выдать **API key** с лимитом расходов — если плана с подпиской не получается оформить (нет карты, банк не пропускает). Напиши Константину (TG @altbar) с обоснованием задач.

## Регистрация подписки

1. Зайди на https://claude.ai (через VPN)
2. Sign up через Google или email
3. Подтверди email
4. После входа — в правом верхнем углу аватар → **Settings** → **Subscription**
5. Выбери план, привяжи карту

## Авторизация в Claude Code

В терминале:

```bash
claude
```

(или `claude --version` сначала, чтобы убедиться что бинарь жив)

При первом запуске Claude спросит, как авторизоваться:

```
How would you like to authenticate?
> Claude account (recommended)
  API key
```

**Выбирай "Claude account"** если у тебя Pro/Max — откроется браузер на claude.ai/auth, залогинишься как обычно. Токен сохранится надолго (около года), повторно логиниться не нужно.

**Выбирай "API key"** только если УК выдала тебе ключ. Тогда вставь его прямо в терминал.

## Если браузер не открывается / login висит

Это та же проблема DPI:
1. Проверь VPN активен
2. Зайди на https://claude.ai в обычном браузере — должно открыться
3. Если открывается в браузере, но Claude Code не открывает — попробуй вручную скопировать URL, который Claude печатает в терминале, и открыть его в браузере с VPN

## Проверка что всё работает

После успешной авторизации в Claude:

```
Привет! Скажи кратко, какая модель меня обслуживает и какой у меня план.
```

Claude должен ответить и упомянуть модель (типа Opus 4.x, Sonnet 4.x). Если ответил — всё работает.

```bash
claude auth status
```

(вне сессии Claude) — покажет твой текущий статус авторизации.

## Что НЕ делать

- **Не запускай `claude auth login` повторно**, если уже залогинен — может перетереть рабочий токен. Если что-то с авторизацией не так — пиши Константину, не дёргай команды наугад.
- **Не пробуй обновить/рефрешить токен скриптами** — Anthropic это не любит, может прилететь блок.
- **Не публикуй API key нигде** — ни в чатах, ни в коде, ни в скриншотах. Утечка = чужой расход твоего бюджета.
