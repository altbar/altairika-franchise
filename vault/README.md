# Vault — материалы от УК

В этой папке управляющая компания Altairika складывает материалы, актуальные для всех франчайзи: сезонные акции, новые программы, методички, нововведения.

## Что здесь будет

- **Сезонные кампании** — НГ, 8 марта, 1 сентября, лето: готовые офферы, тексты, идеи
- **Новые программы** — описание новой VR-программы, целевой возраст, как продвигать
- **Юридические шаблоны** — договор-оферта с родителем, согласие на фото, шаблоны претензий
- **Методички** — "как работать со школами", "как настраивать таргет", "как считать unit-экономику ДР"
- **Новости сети** — анонсы новых партнёров, общесетевые активности, отчёты

## Как читать

Файлы именуются: `YYYY-MM-DD-короткое-описание.md`. Сортировка по дате — свежее наверху.

```
vault/
  2026-04-30-summer-program-launch.md
  2026-04-15-new-ocean-program.md
  2026-04-01-1-september-campaign-template.md
  ...
```

## Как УК добавляет материалы

УК (Константин Урвантцев) пушит изменения в `main` репо `altbar/altairika-franchise`. У всех франчайзи запущенный `altairika-update` подтянет изменения через `git pull`.

После обновления Claude видит новые материалы — можно спросить:
> Что нового в vault'е за последнюю неделю?
> Прочитай vault/2026-04-30-summer-program-launch.md и расскажи что я могу применить у себя.

## Как франчайзи добавлять свои материалы НЕ нужно

Vault — read-only для франчайзи. Если у тебя есть полезный шаблон, который ты считаешь стоит распространить на сеть — отправь Константину (TG @altbar). Если идея зайдёт — он добавит в vault от имени УК.

> Запросы на изменение существующих материалов — также через УК.

## Краткое содержание (1-2 абзаца)

[развёрнутое описание]

[конкретные действия — какие посты опубликовать, что добавить в расписание, какие тексты использовать]

@altbar / k.urvantcev@gmail.com
