# Разбор конкурента

Используется когда хочешь понять, что предлагает конкурент в твоём городе, как он позиционируется, в чём ты можешь быть сильнее или слабее. Требует Exa MCP (Шаг 11).

## Промпт

```
Сделай разбор конкурента: [название организации / ссылка на сайт или соцсеть].

Я — франчайзи Altairika в [город], делаю VR-программы для детей 4-12 лет.

Через exa найди:
1. Сайт конкурента и/или активные соцсети — основное описание услуг
2. Упоминания в локальной прессе / VK-сообществах / 2GIS
3. Отзывы клиентов (скан 2GIS, Я.Карты, отзовиков)
4. Цены, если открыто опубликованы
5. Ассортимент программ и форматов

Сведи в таблицу:

| Параметр | Конкурент | Altairika (моя локация) |
|----------|-----------|--------------------------|
| Целевая аудитория | | |
| Форматы (ДР, школьные программы, индивид. сеансы) | | |
| Цены (если известны) | | |
| Длительность сеанса | | |
| Размер группы | | |
| Уникальные фишки | VR? AR? мастер-классы? | VR + образовательная программа |
| Слабые места по отзывам | | — |
| Сильные места по отзывам | | — |

После таблицы — короткие выводы:
1. В чём конкурент сильнее меня объективно
2. В чём я сильнее
3. Чего у конкурента нет, что я могу запустить
4. Чего у меня нет, чем стоит дополнить ассортимент

Не выдумывай данные — если не нашёл, пиши "нет данных".
```

## Заметки

- Хорошо работает для "моно-конкурентов" — VR-аттракционы, детские развивающие центры, развлекательные парки
- Для общих развлекательных мест (батутные центры, детские кафе) — возьми только верхнеуровневое сравнение по ЦА и форматам
- Не публикуй результат как пост — это для твоего внутреннего понимания
