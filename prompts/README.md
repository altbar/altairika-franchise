# Prompt Templates

Готовые промпты под повторяющиеся задачи. Отличие от скиллов в `skills/`:
- **Скиллы** — Claude применяет автоматически, когда понимает что задача матчится. Содержат методологию.
- **Prompts** — ты сам копипастишь в Claude, заполняешь плейсхолдеры. Прямая команда.

## Как пользоваться

1. Открой нужный файл
2. Скопируй текст внутри ```promot``` блока
3. Замени `[плейсхолдеры в скобках]` на свои данные
4. Вставь в Claude → Enter

## Список

- `weekly-content-plan.md` — план постов на неделю
- `competitor-analysis.md` — разбор конкурента в твоём городе
- `monthly-report.md` — отчёт по месяцу для УК
- `holiday-campaign.md` — праздничная кампания (НГ / 8 марта / 1 сентября)
- `feedback-summary.md` — сводка отзывов за период
