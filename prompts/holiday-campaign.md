# Праздничная кампания

Используется за 2-3 недели до праздника — Новый год, 23 февраля, 8 марта, 1 сентября, Хэллоуин. Claude составит набор материалов: пост-анонс, серию постов прогрева, оффер для родителей, идеи для тематической программы.

## Промпт

```
Составь маркетинговую кампанию к [праздник, например "Новый год 2027"].

Локация: Altairika [город]
Дата запуска кампании: [дата начала продвижения]
Дата самого праздника: [дата]

Что планируется:
- [специальная программа? тематические сеансы? скидка? подарочные сертификаты?]
- [сколько свободных слотов?]

Целевая аудитория: родители 28-45, дети 4-12.

Что нужно:
1. **Пост-анонс** — для TG/VK, объявляющий запуск кампании. Длина 400-600 знаков.
2. **3 поста прогрева** — постепенно за 2 недели до даты, разные углы (что ребёнок получит / отзыв клиента прошлого года / закулисье подготовки)
3. **Оффер-блок** — короткий текст для прикрепления к рассылкам / закрепу: что предлагаем, цена, контакт
4. **Скрипт ответа на типичный вопрос** "осталось ли место на [дата]" — для администратора
5. **Идея для контент-зацепки** — что снять / показать в день праздника, чтобы потом превратить в пост-отчёт

Тон: согласно CLAUDE.md (без эмодзи в массе, без "круто/вау", конкретика).

Эмодзи разрешены только тематические — для Нового года 1 ёлочка ОК, для 8 марта 1 цветок ОК. Без спама.
```

## Заметки

- За 2-3 недели — нормальный цикл. За меньше — не успеваешь раскрутить, за больше — теряется актуальность
- Для Нового года добавь продажу подарочных сертификатов — отдельный продукт со своей экономикой
- Для 1 сентября акцент на "после школы" / "снять стресс первой недели" — не на подарок к школе
- На Хэллоуин — спорный праздник для части ЦА (родители-консерваторы), формулируй мягко: "вечер историй и тайн" вместо "хэллоуин-вечеринка"
