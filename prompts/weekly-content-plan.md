# Контент-план на неделю

Используется в воскресенье/понедельник. Claude составит план постов на 7 дней с учётом твоей загрузки и текущих программ.

## Промпт

```
Составь контент-план постов на неделю с [дата начала] по [дата конца].

Платформы: [TG / VK / Instagram — выбери одну или несколько]
Частота: [сколько постов в неделю на каждой]

Что есть на этой неделе:
- Расписание сеансов: [перечисли свободные слоты, акценты — например, "суббота 14:00 космос для младших, воскресенье 12:00 океан"]
- Мероприятия: [ДР, групповые программы, школьные группы — что и когда]
- Акции: [если есть скидка / акция — описание]

Аудитория: родители 28-45, дети 4-12, [твой город].

Тон: согласно CLAUDE.md (без эмодзи в массе, без "круто/вау", конкретика, что ребёнок получит).

Для каждого поста:
1. День и платформа
2. Тип (анонс / отзыв / расписание / напоминание / закулисье)
3. Готовый текст
4. Призыв к действию
```

## Заметки

- Если не знаешь, что есть на неделе — заполни хотя бы одну строку, Claude сам предложит чем дополнить
- Не давай Claude ФИО клиентов, чьи отзывы хочешь использовать — обезличь до "мама 6-летнего сына"
- После получения плана — пробегись глазами, удали то, что не пойдёт у тебя
