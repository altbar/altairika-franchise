## Что такое Altairika

VR-образовательная франшиза для детей 4-12 лет. Шлемы виртуальной реальности с обучающим контентом — космос, океан, тело человека, история. Формат: 30-45 минутные сессии, групповые программы для школ/садов, дни рождения, семейные мероприятия. Сеть из 73 партнёров в 52 регионах России.

## Целевая аудитория

- **Родители 28-45 лет** с детьми 4-12 — главные ЛПР
- **Школы и детские сады** — групповые образовательные программы
- **Организаторы ДР** — детские дни рождения как отдельный продукт
- **Семейные мероприятия** — выходные, праздники

## Тон бренда

- Тёплый, но экспертный. Родители доверяют — мы образование, не аттракцион.
- Без эмодзи в массе. Допустимо 1-2 на пост в соцсетях, не больше.
- Возраст детей 4-12 — пиши так, чтобы родителю было понятно "что ребёнок там получит, чему научится".
- Никогда: "круто", "вау", "ваще", "мега". Можно: "интересно", "увлекательно", "познавательно".
- Английские слова — минимум. "VR" ок, остальное лучше по-русски.
- Без обращений "дорогие родители" — звучит фальшиво. Лучше "родители" или прямо к ребёнку.

# Технические заметки по Claude Code

## Базовые команды в Claude

- `/help` — список всех команд
- `/clear` — очистить контекст (если разговор стал длинным и Claude путается / медленно отвечает)
- `/compact` — сжать контекст (сохраняет саммари, освобождает память — полезно после длинной сессии)
- `/init` — заполнить CLAUDE.md в текущей папке проекта (если работаешь над конкретной задачей в отдельной папке)
- `/config` — настройки

## Перезапуск, если завис

- Ctrl+C — прервать текущий запрос
- Закрой и открой терминал заново — после установки нового скилла или обновления PATH
- Если совсем глухо: `claude --version` снаружи Claude (просто проверка, что бинарь жив)

## Где что лежит

- **Mac**: `~/.claude/` — конфиги, скиллы, память
- **Windows**: `%USERPROFILE%\.claude\` — то же самое
- **CLAUDE.md** (этот файл) — главная инструкция, Claude её читает в начале каждой сессии
- **profile.yaml** — твои данные (если решишь завести)
- **settings.json** — настройки внешнего вида и поведения

## Статусная строка

Чтобы видеть текущую папку и git-ветку под промптом — открой `~/.claude/settings.json` (Mac) или `%USERPROFILE%\.claude\settings.json` (Windows) и добавь:

```json
{
  "statusLine": {
    "type": "command",
    "command": "echo \"$(pwd) | $(git branch --show-current 2>/dev/null || echo no-git)\""
  }
}
```

Перезапусти Claude — внизу появится статусная строка.

## Voice input — Wispr Flow

Если поставлен Wispr Flow (https://wisprflow.ai):
- Удерживай **Fn** (Mac) / **Right-Alt** (Windows)
- Говори обычным темпом
- Отпусти hotkey — текст вставится в активное окно (включая prompt в Claude)

Длинные промпты (от 3 предложений) проще диктовать. Короткие команды — печатать.

## Обновления кита

Когда УК выкатит обновления (новые скилы, инструкции, шаблоны) — запусти:

```bash
altairika-update
```

Скрипт сделает `git pull` в `~/altairika-franchise/` и подложит обновления в `~/.claude/`. Подробнее: см. README в репо после клонирования.
